# YouTube Clone

#### Demo link: <a href="https://yt-youtube-clone.netlify.app/">https://yt-youtube-clone.netlify.app/</a>

### Tech Stack:
- React
- Tailwind CSS
- Material UI

**To run the app**
```
git clone https://github.com/Hemanthghs/YouTube-Clone-React-Tailwind-MUI

cd YouTube-Clone-React-Tailwind-MUI

npm install

npm start
```
